ReAct = Reason + Action.

AI reasons about the problem and performs tools/actions.

Thought -> Action -> Observation -> Final Result.


Eg:

Whats is the weather in chennai ?

AI:
Think it needs weather data.
Calls weather api
Observes the result
Responds.


Tool Calling.


Prompt:
======

- get_weather[city]: Returns the current weather.

Format:
Thought: [Your reasoning]
Action: [tool_name][argument]
Observation: [Result from tool]
... (repeat until done)
